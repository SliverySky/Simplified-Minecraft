本项目由C++编写，主要用到了OpenGL和SFML。

[√] 天空盒子

[√] 基本光照

[√] 基于深度贴图的阴影渲染

[√] 方块的移除和添加

[√] 不同工具的选择

[√] 基于噪声生成方块地形
