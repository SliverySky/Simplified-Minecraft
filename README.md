### 简化版Minecraft



本项目由C++编写，主要用到了OpenGL和SFML。

#### 目前的进度

[√] 地图基本方块的加载和渲染

[√] 摄像机

[√] 天空盒子

[√] 基本光照

[√] 基于深度贴图的阴影渲染

[√] 方块的移除和添加

[√] 不同建造工具的选择

[√] 基于噪声生成方块地形

[√] 性能优化：视锥体剔除
